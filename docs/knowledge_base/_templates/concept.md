# 概念定义文档模板

> 本模板用于定义心理学、社会学、传播学等领域的专业概念。每个概念文档应包含完整的理论背景、实践应用和案例说明。

## 适用场景

- 定义专业术语或学术概念
- 建立理论知识体系
- 为技能文档提供理论支撑

## 文档结构

### 1. 元数据 (Frontmatter)

```yaml
---
title: [概念名称]
description: [一句话定义，不超过100字]
category: [分类路径，如：01_foundation/心理学基础]
tags: [标签1, 标签2, 标签3]
related_concepts: [相关概念列表]
version: 1.0.0
created_at: YYYY-MM-DD
updated_at: YYYY-MM-DD
difficulty: beginner|intermediate|advanced
target_agent: 情感社交Agent
confidence_level: high|medium|low
sources: [理论来源]
applicability: [适用场景列表]
---
```

## 编写规范

1. **定义准确性**：引用权威学术来源，确保概念定义准确
2. **实践导向**：每个概念都要说明如何在情感社交场景中应用
3. **案例充实**：至少包含2个实际案例说明概念的应用
4. **标签统一**：使用标签体系中的标准标签
5. **交叉引用**：建立概念间的关联关系

## 注意事项

- 定义部分要简洁明了，一句话能概括核心
- 理论基础要引用真实的学术来源
- 实践应用要具体可操作，避免空泛描述
- 常见误区要指出实际操作中的陷阱

## 占位符说明

本模板中的占位符采用双大括号语法，复制模板后请统一替换：

- `{{title}}`：文档标题
- `{{description}}`：一句话摘要（≤100字）
- `{{category}}`：所属分类路径
- `{{tags}}`：标签数组，遵循 `00_meta/01_标签体系.md`
- `{{author}}`：作者或团队
- `{{created_at}}` / `{{updated_at}}`：日期，格式 `YYYY-MM-DD`
- `{{body}}`：正文主体，按下方结构逐节填写
