# Q&A 问答文档模板

> 本模板用于编写常见问题解答文档。每个文档应聚焦特定主题，收集高频问题并提供专业解答。

## 适用场景

- 特定主题的高频问题汇总
- 场景化问题解答
- 情境应对策略
- 用户常见困惑解答

## 编写规范

1. **问题代表性**：确保收集的问题是真正高频的
2. **分析深度**：不仅给出答案，还要解释为什么
3. **话术实用**：提供可以直接使用的话术模板
4. **情景具体**：每个问题都要有具体的情境描述
5. **避免雷同**：不同问题之间要有差异化，避免重复

## 问题收集标准

| 标准 | 说明 |
|------|------|
| 高频性 | 在实际场景中反复出现 |
| 代表性 | 能代表一类相关问题 |
| 可解答 | 有明确的解决方案 |
| 实用价值 | 解决后能产生实际效果 |

## 注意事项

- 一个主题文档建议包含5-10个问题
- 问题按重要性或发生频率排序
- 每个问题要有完整的分析和建议
- 建立问题之间的关联关系

## 占位符说明

本模板中的占位符采用双大括号语法，复制模板后请统一替换：

- `{{title}}`：文档标题
- `{{description}}`：一句话摘要（≤100字）
- `{{category}}`：所属分类路径
- `{{tags}}`：标签数组，遵循 `00_meta/01_标签体系.md`
- `{{author}}`：作者或团队
- `{{created_at}}` / `{{updated_at}}`：日期，格式 `YYYY-MM-DD`
- `{{body}}`：正文主体，按下方结构逐节填写
