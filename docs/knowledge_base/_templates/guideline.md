# 指南规范文档模板

> 本模板用于编写技能操作指南、最佳实践、步骤流程等文档。每个指南文档应包含清晰的步骤、具体的示例和可操作的建议。

## 适用场景

- 技能操作指南（如：如何开场、如何倾听）
- 最佳实践总结（如：关系维护、冲突处理）
- 步骤流程说明（如：破冰步骤、信任建立流程）
- 判断标准指南（如：如何判断关系阶段）

## 编写规范

1. **步骤清晰**：每个步骤都有明确的目标和操作要点
2. **示例丰富**：提供真实的对话示例和场景案例
3. **可操作性**：每个建议都应该是可以直接执行的
4. **错误预防**：明确指出常见错误和禁忌行为
5. **效果可测**：提供评估指标和自检清单

## 注意事项

- 话术模板要提供多个变体，避免机械重复
- 步骤之间要有逻辑衔接
- 要考虑不同情境的适配性
- 错误案例要和成功案例对比说明

## 占位符说明

本模板中的占位符采用双大括号语法，复制模板后请统一替换：

- `{{title}}`：文档标题
- `{{description}}`：一句话摘要（≤100字）
- `{{category}}`：所属分类路径
- `{{tags}}`：标签数组，遵循 `00_meta/01_标签体系.md`
- `{{author}}`：作者或团队
- `{{created_at}}` / `{{updated_at}}`：日期，格式 `YYYY-MM-DD`
- `{{body}}`：正文主体，按下方结构逐节填写
