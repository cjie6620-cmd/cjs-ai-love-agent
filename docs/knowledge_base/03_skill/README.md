# 03_skill ｜ 技能手册层

> 技能 = 可训练、可迁移、可复用的**行为能力**。本目录把亲密关系中最常用的软技能拆解为可执行的步骤与话术模板，供 AI Agent 在多场景下组合调用。

---

## 一、技能体系总图

```
               ┌── 自我认知   （我是谁 / 我要什么）
               │
  心理建设 ────┤── 情绪管理   （我如何处理情绪）
               │
               └── 情感处理   （我如何处理亲密情感）
                       │
                       ▼
               ┌── 沟通技巧   （我如何传递信息）
 对外互动 ────┤── 社交礼仪   （我如何得体呈现）
               │
               ├── 人际关系   （我如何维护关系）
               │
               └── 冲突处理   （我如何化解矛盾）
```

| 目录 | 技能焦点 | 代表文档 |
|------|----------|----------|

## 三、技能组合模式

| 典型问题 | 建议技能组合 |
|----------|--------------|
| 刚认识怎么聊 | 沟通（提问 + 倾听）+ 礼仪 + 自我披露（`01_foundation/心理学基础/04_自我披露.md`） |
| 女朋友生气了 | 情绪管理（识别）+ 情感处理（共情）+ 沟通（非暴力表达） |
| 被 PUA 了 | 自我认知（核心自尊）+ 心理建设（韧性）+ 人际关系（边界设定） |
| 不会拒绝亲戚介绍 | 沟通（拒绝）+ 人际关系（边界）+ 礼仪（得体措辞） |

---

## 四、技能训练建议

- **刻意练习**：每项技能选 1–2 条话术模板，在真实场景中复盘对比。
- **记录-反思-迭代**：记录真实对话文本，事后对照技能要点找差距。
- **角色扮演**：AI Agent 可提供场景化 role-play，帮助用户脱敏与演练。
- **长期复利**：沟通与情绪技能是复利型能力，6 个月以上持续练习才有显著提升。
