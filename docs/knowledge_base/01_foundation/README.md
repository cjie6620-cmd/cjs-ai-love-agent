# 01_foundation ｜ 理论基础层

> 本目录是整个知识库的"学术根基"，沉淀**传播学、心理学、社会学**三大学科中与亲密关系密切相关的核心理论。上层目录（场景、技能、策略、专题）的所有建议，最终都可回溯到本目录中的一条或多条理论。

---

## 一、为什么需要理论基础

- **可解释性**：当 AI Agent 给出建议时，可通过本目录的理论做出因果解释，避免"空话术"。
- **可泛化性**：基于底层理论的结论能够跨场景迁移，而非停留在个案经验。
- **安全性**：部分建议（如创伤修复、情绪调节）必须基于心理学共识，才能避免二次伤害。

## 三、学科之间的联结

```
心理学（个体内）  ──➜  解释动机、情绪、依恋与偏差
        ↓
传播学（个体间）  ──➜  解释信息如何编码、传递、反馈
        ↓
社会学（关系网）  ──➜  解释关系在群体与文化中的运行
```

上述三层递进构成了分析亲密关系问题的完整框架：**先看内在（心理），再看交互（传播），最后看结构（社会）**。

## 四、使用建议

- **理论 + 场景双召回**：回答场景类问题时，应同时召回 `02_scenario/` 与本目录的相关理论，回答深度更足。
- **引用规范**：引用理论时尽量带上原作者与年份，例：*（Bowlby, 1969）*。
- **避免过度抽象**：将理论作为思路引导，而非原文照搬抛给用户。
