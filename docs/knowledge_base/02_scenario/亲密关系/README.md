# 亲密关系

> 亲密关系是个人福祉最重要的预测因子之一（Harvard Grant Study, 2017）。本目录聚焦**恋人关系**的全生命周期管理，提供从"单身期自我提升"到"分手后修复"的系统方法论。

---

## 一、关系生命周期模型

```
单身期 ─➜ 相识期 ─➜ 暧昧期 ─➜ 确认期 ─➜ 热恋期 ─➜ 平淡期 ─➜ 稳定期
                                                ↘          ↙
                                                  冲突期
                                                      ↓
                                            修复 OR 分手 ─➜ 恢复期
```

每个阶段都有独立的关键任务与典型困境，见 `恋人关系/` 子目录中 8 篇核心文档。

## 三、核心理论支撑

- **斯滕伯格爱情三角理论**：激情 + 亲密 + 承诺，见 `01_foundation/心理学基础/`。
- **Gottman 四骑士模型**：批评、蔑视、防御、筑墙——预测关系破裂的四个信号。
- **依恋理论**：理解伴侣行为模式的底层钥匙。
- **社会渗透理论**：自我披露由浅入深推动亲密度提升。

## 四、使用建议

- 遇到具体问题时，先判断所处生命周期阶段，再检索对应文档。
- 长期关系问题几乎都可回溯到"沟通 + 情绪调节 + 共同愿景"三个基本面。
- 高风险信号（冷暴力、出轨、家暴）直接跳转至 `relationship_knowledge/11_安全红线与禁忌话术.md`。
