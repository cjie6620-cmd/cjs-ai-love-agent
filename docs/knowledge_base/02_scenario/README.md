# 02_scenario ｜ 场景指南层

> 本目录按**现实生活中的社交场景**组织知识，解决"在 X 场景下我该怎么办"这类高频问题。每个子目录都遵循「场景识别 → 关键挑战 → 应对原则 → 操作步骤 → 话术示例」的结构。

---

## 一、场景分层逻辑

按**关系亲密度**与**角色权力距离**两个维度划分四大场景：

| 亲密度\权力 | 对等 | 不对等 |
|-------------|------|--------|
| **高** | `亲密关系/` | `家庭关系/`（代际差） |
| **低** | `社交拓展/` | `职场社交/`（上下级） |

---

## 二、子目录索引

### 2.1 `亲密关系/恋人关系/`
覆盖恋爱全生命周期：单身期 → 相识 → 约会 → 确认关系 → 热恋 → 平淡 → 长期 → 分手修复。

### 2.2 `家庭关系/`
原生家庭认知、与父母沟通、兄弟姐妹、婆媳翁婿、家庭聚会。

### 2.3 `社交拓展/`
自我介绍、融入陌生圈子、老朋友维护、优质人脉拓展、线上礼仪、线下活动。

### 2.4 `职场社交/`
新人入门、上下级关系、跨部门协作、八卦应对、职场友谊边界、离职后关系。

---

## 三、场景问答的召回策略

1. **识别场景关键词**（见家长 / 相亲 / 同事告白 / 分手挽回 …）。
2. **定位一级目录**（家庭 / 社交 / 职场 / 亲密）。
3. **命中具体文档**，并二路召回 `03_skill/` 中的沟通、情绪、冲突相关技能。
4. **必要时三路召回** `01_foundation/` 做学理解释。

---

## 四、通用场景应对原则

- **先情绪后事件**：任何场景问题，先识别情绪并回应，再讨论事件对策。
- **尊重差异**：不同地域、代际、文化背景的人对同一场景的期待可能完全相反。
- **保留余地**：短期话术要考虑中长期关系影响，避免一次性"爽快话"。
- **安全兜底**：涉及暴力、骚扰、胁迫场景，优先建议用户远离并寻求社会支持。
