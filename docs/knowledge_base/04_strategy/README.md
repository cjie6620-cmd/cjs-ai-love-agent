# 04_strategy ｜ 策略指南层

> 策略 = 针对特定关系目标的**有序行为组合**。相较于 `03_skill/` 的单项技能，本目录提供的是"连招"与"全局战术"，回答"为了推进 X 目标，我该按什么顺序做什么"。

---

## 一、策略分层

```
短期战术（一次对话 / 一次约会）         → 破冰技巧、沟通技巧
中期战役（数周到数月的关系阶段）         → 信任建立、关系推进、冲突处理、边界表达
长期战略（一年以上的共同生活与承诺）     → 关系深化
```

---

## 二、子目录索引

| 目录 | 策略目标 | 关键问题 |
|------|----------|----------|
| `破冰技巧/` | 消除初次接触的陌生感 | 怎么开场、聊什么、如何延续 |
| `信任建立/` | 让对方愿意让渡脆弱 | 第一印象、一致性、可预期性 |
| `关系推进/` | 从普通朋友到恋人 | 升级节奏、表白时机、承诺表达 |
| `沟通技巧/` | 深度对话与共鸣 | 从闲聊到情感披露 |
| `冲突处理/` | 把冲突变成关系增长点 | Gottman 修复尝试、冷静-复盘-承诺 |
| `关系深化/` | 长期稳定与共同愿景 | 共同叙事、仪式感、成长同步 |
| `边界表达/` | 在亲密中保持独立 | 拒绝、需求表达、边界维护 |

---

## 三、推进策略的一般节奏

1. **破冰**（0–2 次接触）：降低防御、制造正向情绪。
2. **建立信任**（2–8 次接触）：保持言行一致、适度自我披露。
3. **升温**（1–3 个月）：共享体验、制造共同回忆。
4. **DTR 确认**（时机成熟时）：正式定义关系，避免"吊着对方"。
5. **深化**（6 个月以上）：共同规划、处理冲突、保持亲密仪式。

> **原则**：节奏可快可慢，但方向必须清晰。模糊期拖得越久，越容易滑入"消耗型关系"。

---

## 四、策略落地建议

- **先看阶段，再选策略**：错用策略（例如在刚认识时就谈未来）往往适得其反。
- **策略服务于诚意**：本目录不是"PUA 手册"，一切策略的前提是真实的尊重与善意。
- **可回撤**：任何策略都要给自己与对方留退出空间，避免走向情感绑架。
- **与技能配合**：策略提供"做什么"，技能提供"怎么做"，二者需要配套调用。
