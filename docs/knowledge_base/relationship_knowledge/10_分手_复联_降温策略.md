# 分手、复联、降温策略

> 这份文档用于帮助模型处理分手后的高情绪场景，重点是降温、止损、体面和恢复现实感。

## REL-BREAKUP-001 刚分手时，我特别想疯狂发消息，怎么办？

### Q：刚分手两天，我整个人都很乱，特别想一直给她发消息解释、认错、挽回。我知道这样可能不好，但真的忍不住。这个时候怎么办？

### A：
刚分手时最重要的，不是追回来，而是先别让局面因为冲动变得更糟。

人在刚分手时常见的冲动有：
- 想立刻解释清楚
- 想马上证明自己会改
- 想靠连续输出把关系拉回来

但这时候对方通常也在高情绪里，你的高频输出更容易被听成施压，而不是诚意。

更稳的做法：
- 先停掉轰炸式联系
- 先处理睡眠、吃饭、现实秩序
- 把最想说的话先写下来，不立刻发

推荐表达：
- “我知道现在情绪都很重，我先不在这个时候逼你回应。”
- “我会先把状态稳住，不想在最乱的时候继续伤害彼此。”

避免表达：
- “你现在就把话说清楚。”
- “我不会停，直到你回来。”

检索关键词：
- 刚分手想发很多消息
- 分手后别轰炸
- 怎么先降温
- 分手初期怎么办

## REL-BREAKUP-002 我满脑子都是复合，什么都做不了，怎么先把自己拉回来？

### Q：分手后我现在脑子里只有一个念头，就是复合。工作也做不了，饭也吃不下。怎么先把自己拉回一点？

### A：
当你整个人只剩“我要复合”一个念头时，说明你现在最需要的不是行动策略，而是先止住情绪失控。

你可以先做的事：
- 把今天缩小到最基本的生活任务，比如吃饭、洗澡、出门走十分钟
- 找一个现实中的人说一说，而不是只在脑子里循环
- 暂时不要把“复合”当作今天必须解决的任务

更关键的是认清：你现在想复合，可能不只是因为真的想清楚了，而是因为失去感太痛，急着找一个止痛方法。

推荐表达：
- “我现在先不逼自己做关系决定，先把今天撑过去。”
- “我现在最需要的是把自己从这股冲动里拉出来一点。”

避免表达：
- “没有她我活不下去。”
- “我现在就要把她追回来，不然我什么都做不了。”

检索关键词：
- 分手后控制不住想复合
- 脑子里全是前任
- 先稳住自己
- 分手后止痛

## REL-BREAKUP-003 什么时候才适合复联？

### Q：我不是完全放不下，但也怕自己只是上头。一般要满足什么条件，才算适合复联？

### A：
适不适合复联，不是看你想不想，而是看现在有没有基础。

比较重要的判断点：
- 分手原因你有没有真正想清楚
- 你联系她，是为了认真沟通，还是为了缓解痛苦
- 对方有没有表达过明确拒绝和不想被打扰
- 你能不能接受复联后不一定复合这个结果

如果问题本身没变、边界也没处理、你还承受不了任何结果，那通常还不适合复联。

推荐表达：
- “我得先确定这次联系不是为了止痛，而是真的有成熟沟通的基础。”

避免表达：
- “只要我还喜欢，就一定值得再试。”
- “先联系了再说，之后再想后果。”

检索关键词：
- 什么时候适合复联
- 复联判断标准
- 分手后还能联系吗
- 复合前要看什么

## REL-BREAKUP-004 复联第一句怎么发，才体面又不吓人？

### Q：如果真的决定复联，第一句到底怎么发？我怕太冷像陌生人，太重又像施压。

### A：
复联第一句的目标只有一个：打开一个低压力窗口，而不是一次性拿回关系。

所以复联开场要满足：
- 轻
- 短
- 可进可退

推荐表达：
- “最近想到一件之前没说清的事，想找个合适的时候和你补一句。如果你不方便，也没关系。”
- “前阵子看到一个你会想到的东西，顺手想起你，祝你最近还顺利。”

避免表达：
- “我每天都在后悔，我们重新开始吧。”
- “你为什么还不回我，我真的快撑不住了。”

检索关键词：
- 复联第一句怎么发
- 前任重新联系开场
- 怎么体面复联
- 分手后第一条消息

## REL-BREAKUP-005 对方明确拒绝了，我怎么收手才不继续伤自己？

### Q：我已经试过联系，她也明确拒绝了。可我还是会忍不住幻想还有机会。怎么收住自己？

### A：
当对方已经明确拒绝后，最重要的不是再想“还能不能翻盘”，而是停止让自己反复停在同一个伤口里。

你可以先做这些事：
- 把“她拒绝我”从“我不值得被爱”里分开
- 暂停通过她的动态和状态反复刺激自己
- 允许自己失落，但别继续拿联系当止痛药

推荐表达：
- “我尊重你的决定，也会把这一步停下来。”
- “我知道继续追下去只会让彼此更累，我先把边界收住。”

避免表达：
- “我再最后努力一次。”
- “你只是现在想不通，我等你。”

检索关键词：
- 被前任拒绝怎么办
- 明确拒绝后怎么收手
- 分手后放不下
- 停止幻想复合

## REL-BREAKUP-006 看到前任动态、想到过去就崩，怎么缓？

### Q：我本来好一点了，但一看到她的动态、想到以前的事，情绪就一下子掉下去。怎么缓这种反复？

### A：
分手恢复不是直线，反复是正常的。问题不在于你又难过了，而在于每次触发后你怎么照顾自己。

更有效的处理方式：
- 承认触发带来的情绪，而不是骂自己没出息
- 降低接触刺激源的频率
- 给自己一个固定的回稳动作，比如出门走路、找朋友、写下此刻想法

推荐表达：
- “我现在被勾起来了，但这不代表我白熬了，只是说明这个伤口还在恢复。”
- “我先不继续刺激自己，先把状态从最难受的点拉回来。”

避免表达：
- “我怎么还这么废。”
- “我要去找她，不然我受不了。”

检索关键词：
- 看到前任动态崩了
- 分手后反复
- 如何缓解触发
- 走出分手

## 场景分类

- 按关系阶段：陌生 / 破冰 / 暧昧 / 确定 / 稳定 / 降温 / 分手。
- 按情绪状态：平稳 / 焦虑 / 愤怒 / 低落 / 危机。
- 按沟通介质：文字 / 语音 / 视频 / 线下。

## 触发信号

- 行为信号：回复节奏变化、主动性下降、话题深度变化。
- 语言信号：出现"随便""都行""没事"等低能量词。
- 情绪信号：突然冷淡、突然粘人、情绪波动剧烈。

## 话术模板矩阵

| 原句（常见表达） | 优化句（建议话术） | 优化点 |
|------------------|---------------------|--------|
| "你怎么还不回我" | "刚刚看到你没回，是不是在忙？" | 把指责换成关心 |
| "随便你" | "我现在有点情绪，等我缓一下再聊好吗" | 把冷暴力换成边界表达 |
| "你根本不在乎我" | "我最近有点没安全感，我需要你多回应一点" | 把评判换成需求 |
| "算了不说了" | "这件事我还没想清楚，我们晚点再聊" | 把关闭换成延后 |

## 使用注意

- 话术需匹配关系阶段与双方熟悉度，避免越位。
- 不要把模板句作为"标准答案"机械使用。
- 情绪高峰期优先降温，不要在愤怒中输出关键决策。

## 红线

- 禁止贬低、羞辱、人格攻击类表达。
- 禁止以分手/冷战作为操控手段。
- 涉及安全风险，跳转 `11_安全红线与禁忌话术.md`。
