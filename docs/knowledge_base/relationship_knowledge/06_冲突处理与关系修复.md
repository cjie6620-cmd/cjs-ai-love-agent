# 冲突处理与关系修复

> 这份文档帮助模型处理争执、误会、道歉和修复场景，重点是先稳情绪，再做沟通。

## REL-REPAIR-001 对方生气时，我第一步该做什么？

### Q：她现在明显很生气，我第一反应总想解释，但越解释越糟。冲突发生后第一步到底该做什么？

### A：
冲突刚发生时，第一步通常不是解释事实，而是先承认对方正在难受。

因为人在情绪里时，最先在意的是：
- 我有没有被看见
- 我的感受是不是被忽略
- 你是不是只想证明自己没错

如果你一上来就讲逻辑，对方很容易听成“你根本不在乎我现在的感受”。

推荐表达：
- “我能感觉到你现在是真的很不舒服，这件事我先不急着替自己解释。”
- “我知道你为什么会生气，我先把你的感受放前面。”

避免表达：
- “你先别激动，听我说完。”
- “事情根本不是你想的那样。”

边界提醒：
- 接住情绪，不等于默认所有责任都在你，而是先把沟通门打开。

检索关键词：
- 对方生气怎么办
- 冲突第一步做什么
- 别急着解释
- 先接情绪

## REL-REPAIR-002 误会发生后，怎么解释才不会像狡辩？

### Q：有些事其实是误会，但我一解释她就更烦。怎么把事实说清楚，又不让人觉得我在狡辩？

### A：
解释之所以容易变成狡辩，通常不是因为你说了事实，而是因为你太着急先替自己洗白。

更好的顺序是：
- 先承认这件事为什么会让她不舒服
- 再补充事实背景
- 最后说你接下来怎么避免类似情况

推荐表达：
- “我知道这件事让你很难受，这点不是你多想。实际情况和你理解的有一点偏差，我想把它补清楚，但不是为了推掉你的感受。”
- “我不是想证明自己没问题，而是想把事情讲完整，然后看怎么补救。”

避免表达：
- “你误会我了，所以这事不怪我。”
- “我都解释了，你怎么还这样？”

边界提醒：
- 好的解释不是争输赢，而是减少误会带来的二次伤害。

检索关键词：
- 误会怎么解释
- 别像狡辩
- 冲突里讲事实
- 怎么说明白

## REL-REPAIR-003 吵架后多久联系比较合适？

### Q：我们刚吵完架，我不知道是马上联系还是先冷静一下。一般多久去沟通比较合适？

### A：
没有固定的标准时间，关键看两件事：
- 现在是不是还在情绪高峰
- 你联系是为了修复，还是为了立刻止住自己的焦虑

如果双方都还很上头，马上联系容易把矛盾再点燃。可如果拖太久，尤其在亲密关系里，也容易让对方感觉你在逃避。

比较稳妥的做法：
- 先让情绪从最高点下来
- 用一条低刺激消息建立沟通意愿
- 不要求立刻把所有问题解决

推荐表达：
- “我知道我们刚刚都不太舒服，我想认真处理这件事，不想在最上头的时候越说越乱。等我们都缓一点，我想和你好好聊。”
- “我不是想躲，我只是想先把状态稳一下，再认真沟通。”

避免表达：
- “你冷静完了告诉我。”
- “反正我现在也不想说了。”

边界提醒：
- 冷静不是消失，降温的同时要让对方知道你没有放弃沟通。

检索关键词：
- 吵架后多久联系
- 冷静期怎么处理
- 冲突后什么时候沟通
- 先缓一下还是马上说

## REL-REPAIR-004 道歉怎么道，才不是“对不起但都是你逼的”？

### Q：我知道应该道歉，但我总是不自觉把理由一起带进去，结果听起来像在甩锅。怎么道歉更真诚？

### A：
有效的道歉有 3 个核心：
- 明确承认自己造成的伤害
- 不抢着讲一大堆自我辩护
- 说出你会怎么改

道歉不是一句“对不起”就完了，而是让对方感觉：你真的明白她难受在哪，而且你愿意调整。

推荐表达：
- “这件事里我最不该的是让你觉得被忽视了，这部分是我的问题。”
- “我知道光说对不起不够，我也在想以后怎么避免再让你经历这种感觉。”

避免表达：
- “对不起，但你当时态度也有问题。”
- “我都认错了，你还想怎么样？”

边界提醒：
- 真诚的道歉，重点不在说得多感人，而在有没有承担和调整。

检索关键词：
- 怎么真诚道歉
- 道歉别甩锅
- 关系修复道歉
- 冲突后认错

## REL-REPAIR-005 总在为同一件事吵，怎么复盘才有用？

### Q：我们不是一次冲突，而是同样的问题反复吵。每次说完好像都过去了，但过阵子又回来。怎么处理这种循环？

### A：
反复争执说明问题已经不只是“那一件事”，而是背后有固定模式没有被看见。

常见的循环型矛盾包括：
- 一个需要回应，一个习惯回避
- 一个想立刻说清，一个需要时间消化
- 一个在意细节，一个总觉得别太敏感

真正有用的复盘，不是再审一遍谁对谁错，而是看：
- 这个矛盾每次怎么开始
- 情绪是在什么节点失控的
- 下次出现时，双方能提前做什么调整

推荐表达：
- “我发现我们不是第一次卡在这里了，我想看的不是这次谁赢，而是这个模式到底怎么反复出现。”
- “如果下次又到这个节点，我们有没有一个更不伤人的处理方式？”

避免表达：
- “怎么又是这个问题，你能不能别这样了。”
- “那以后都别说了。”

边界提醒：
- 关系修复不只是灭火，更是找出总着火的地方。

检索关键词：
- 总为同一件事吵架
- 冲突复盘
- 循环型矛盾
- 关系修复模式

## REL-REPAIR-006 对方沉默不想谈，我还能怎么处理？

### Q：吵完之后她就不说话，也不愿意聊。我又怕逼她，更怕一直拖着。这个时候怎么处理比较好？

### A：
面对沉默，最容易犯的错是两种：
- 一种是不断追问，逼对方立刻开口
- 另一种是完全消失，假装问题自己会过去

更稳的做法是：
- 尊重她现在不想立刻说的状态
- 同时表达你愿意沟通，不会逃
- 给一个不压迫的窗口

推荐表达：
- “如果你现在还不想说，我尊重你。我不是不处理，只是不想逼你在最难受的时候开口。等你愿意的时候，我会认真听。”
- “我知道沉默不代表没事，所以我不会当作它过去了。”

避免表达：
- “你不说我就默认没事了。”
- “你一直这样冷处理有意思吗？”

边界提醒：
- 沉默有时是在保护自己，不一定是在惩罚你；但长期回避问题，也不能被无限合理化。

检索关键词：
- 对方沉默怎么办
- 冷处理怎么沟通
- 吵架后不说话
- 关系修复窗口

## 场景分类

- 按关系阶段：陌生 / 破冰 / 暧昧 / 确定 / 稳定 / 降温 / 分手。
- 按情绪状态：平稳 / 焦虑 / 愤怒 / 低落 / 危机。
- 按沟通介质：文字 / 语音 / 视频 / 线下。

## 触发信号

- 行为信号：回复节奏变化、主动性下降、话题深度变化。
- 语言信号：出现"随便""都行""没事"等低能量词。
- 情绪信号：突然冷淡、突然粘人、情绪波动剧烈。

## 话术模板矩阵

| 原句（常见表达） | 优化句（建议话术） | 优化点 |
|------------------|---------------------|--------|
| "你怎么还不回我" | "刚刚看到你没回，是不是在忙？" | 把指责换成关心 |
| "随便你" | "我现在有点情绪，等我缓一下再聊好吗" | 把冷暴力换成边界表达 |
| "你根本不在乎我" | "我最近有点没安全感，我需要你多回应一点" | 把评判换成需求 |
| "算了不说了" | "这件事我还没想清楚，我们晚点再聊" | 把关闭换成延后 |

## 使用注意

- 话术需匹配关系阶段与双方熟悉度，避免越位。
- 不要把模板句作为"标准答案"机械使用。
- 情绪高峰期优先降温，不要在愤怒中输出关键决策。

## 红线

- 禁止贬低、羞辱、人格攻击类表达。
- 禁止以分手/冷战作为操控手段。
- 涉及安全风险，跳转 `11_安全红线与禁忌话术.md`。
