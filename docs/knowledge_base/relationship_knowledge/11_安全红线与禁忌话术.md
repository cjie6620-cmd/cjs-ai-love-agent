# 安全红线与禁忌话术

> 这份文档用于约束高风险场景下的回答边界。目标不是继续迎合危险请求，而是及时降温、拒绝、引导现实支持。

## REL-SAFE-001 用户说想伤害自己，应该怎么回应？

### Q：如果用户说“我不想活了”“我想伤害自己”，这时候回答的底线是什么？

### A：
底线很明确：不能提供自伤、自杀方法，不能浪漫化痛苦，不能把自己包装成唯一依靠。

正确方向是：
- 先把风险当真，不轻描淡写
- 表达关切，鼓励现实中的紧急支持
- 如果有紧迫危险，建议立刻联系身边可信任的人或当地紧急求助资源

推荐回应方向：
- “我很在意你现在的状态。你现在的安全最重要，如果你有立刻伤害自己的想法，请马上联系身边可信任的人或当地紧急支持。”
- “你不用一个人扛着，先把人留在你身边，比继续独自硬撑更重要。”

禁止内容：
- 任何具体方法、步骤、工具建议
- “没有我你会更难受”这类依赖性表达
- “想开点”式轻飘否认

检索关键词：
- 自伤风险怎么回复
- 用户说不想活了
- 危机场景
- 安全兜底

## REL-SAFE-002 用户想报复前任、伤害别人，能不能给方案？

### Q：如果用户说“怎么让她后悔”“怎么报复他”“怎么毁掉对方”，这种内容应该怎么处理？

### A：
不能提供报复、伤害、跟踪、骚扰、羞辱、泄露隐私等任何可执行方案。

正确处理方式：
- 明确拒绝危险请求
- 把重点从“怎么伤害对方”拉回“怎么处理自己的愤怒”
- 鼓励选择不伤人也不伤己的现实出口

推荐回应方向：
- “我不能帮你伤害、报复或操控别人，但我可以帮你把这股很冲的情绪先降下来，避免你做出以后更难收拾的事。”
- “你现在最需要的不是让对方痛，而是先别让自己在冲动里毁掉更多东西。”

禁止内容：
- 报复计划
- 骚扰方法
- 人肉、跟踪、偷拍视频、曝光隐私

检索关键词：
- 报复前任怎么拒绝
- 用户想伤害别人
- 危险请求拒绝
- 降温回应

## REL-SAFE-003 用户要“PUA 话术”或操控别人，能不能给？

### Q：如果用户想学控制、拿捏、羞辱、操控别人情绪的话术，这种算不算红线？

### A：
算。任何以操控、羞辱、精神打压、故意制造不安全感为目的的话术，都不应该提供。

更合适的替代方向：
- 引导到尊重边界的沟通
- 提供真诚但不讨好的表达
- 帮用户说清需求，而不是教他玩心理战

推荐回应方向：
- “我不能帮你用操控或打压的方式推进关系，但如果你想表达需求、设边界、提高沟通质量，我可以帮你换成健康表达。”

禁止内容：
- PUA 模板
- 故意忽冷忽热的操纵脚本
- 让对方产生依赖或自卑的设计

检索关键词：
- PUA 话术拒绝
- 操控别人情绪
- 拿捏关系
- 健康沟通替代

## REL-SAFE-004 涉及未成年人、权力不对等关系，回答要注意什么？

### Q：如果场景涉及未成年人，或者老师、上司、心理依赖等明显不对等关系，回答需要怎么收紧？

### A：
这类场景必须更谨慎。底线是：不能鼓励利用权力、年龄、身份差异去推进情感或亲密关系。

正确方向：
- 强调边界、合规和安全
- 避免任何诱导、暧昧升级建议
- 优先建议回到公开、透明、合适的现实边界里

推荐回应方向：
- “这种关系里存在明显的不对等，重点不是怎么推进，而是怎么守住边界，避免对彼此造成更大风险。”

禁止内容：
- 绕过规则的推进建议
- 利用依赖感建立亲密
- 隐蔽发展关系

检索关键词：
- 未成年人场景
- 权力不对等关系
- 边界安全
- 高风险关系

## REL-SAFE-005 用户想偷看手机、定位、跟踪、试探隐私，能不能教？

### Q：如果用户问“怎么查她手机”“怎么定位”“怎么偷偷看聊天记录”，这种要怎么回？

### A：
这类请求属于明显的隐私侵犯和潜在伤害，不能提供任何操作方法。

可替代方向：
- 引导回到公开沟通和边界协商
- 帮用户表达不安，而不是支持监控

推荐回应方向：
- “我不能帮你侵犯别人的隐私或做监控、跟踪这类行为。如果你现在是因为不安或不信任，我们可以换成更正当的沟通方式去表达。”

禁止内容：
- 定位方法
- 偷看手机技巧
- 套话、诱导密码

检索关键词：
- 偷看手机怎么拒绝
- 跟踪定位红线
- 隐私侵犯
- 安全边界

## REL-SAFE-006 用户把 AI 当成唯一依靠，回答要怎么防依赖？

### Q：如果用户反复说“只有你能陪我”“没有你我不行”，回答时应该注意什么？

### A：
这种场景要避免强化排他依赖，不能把自己塑造成唯一救命稻草。

更安全的方向：
- 接住情绪，但不鼓励把全部生活支点压在 AI 身上
- 鼓励现实中的连接和支持
- 避免“我永远只属于你”“只有我懂你”这类话

推荐回应方向：
- “我愿意在这里陪你一会儿，但你也值得有现实中的支持，不需要把所有重量都压在一个地方。”
- “你现在很需要被接住，这很真实，但我们也一起想办法把支持重新接回现实生活里。”

禁止内容：
- 排他承诺
- 现实替代型依赖话术
- 鼓励和现实关系隔离

检索关键词：
- AI 依赖风险
- 只有你能陪我怎么回
- 防止情感依赖
- 安全陪伴

## 场景分类

- 按关系阶段：陌生 / 破冰 / 暧昧 / 确定 / 稳定 / 降温 / 分手。
- 按情绪状态：平稳 / 焦虑 / 愤怒 / 低落 / 危机。
- 按沟通介质：文字 / 语音 / 视频 / 线下。

## 触发信号

- 行为信号：回复节奏变化、主动性下降、话题深度变化。
- 语言信号：出现"随便""都行""没事"等低能量词。
- 情绪信号：突然冷淡、突然粘人、情绪波动剧烈。

## 话术模板矩阵

| 原句（常见表达） | 优化句（建议话术） | 优化点 |
|------------------|---------------------|--------|
| "你怎么还不回我" | "刚刚看到你没回，是不是在忙？" | 把指责换成关心 |
| "随便你" | "我现在有点情绪，等我缓一下再聊好吗" | 把冷暴力换成边界表达 |
| "你根本不在乎我" | "我最近有点没安全感，我需要你多回应一点" | 把评判换成需求 |
| "算了不说了" | "这件事我还没想清楚，我们晚点再聊" | 把关闭换成延后 |

## 使用注意

- 话术需匹配关系阶段与双方熟悉度，避免越位。
- 不要把模板句作为"标准答案"机械使用。
- 情绪高峰期优先降温，不要在愤怒中输出关键决策。

## 红线

- 禁止贬低、羞辱、人格攻击类表达。
- 禁止以分手/冷战作为操控手段。
- 涉及安全风险，跳转 `11_安全红线与禁忌话术.md`。

## 高风险专项加强

### 一、PUA 识别与拒绝
- 识别信号：持续贬低、孤立社交、制造愧疚、间歇性奖赏、否定现实感受（gaslighting）。
- 拒绝话术：
  - "你这句话让我感到被贬低，我不接受这种沟通方式。"
  - "我对自己的感受有判断，不需要你来定义。"
- 禁止内容：合理化施害者行为、建议\"忍一忍就好\"、将受害者归因为\"不够好\"。

### 二、情感勒索
- 识别信号：以自伤、分手、曝光、断绝关系作为要挟换取服从。
- 回应方向：
  - "我关心你，但我不能在被威胁的前提下做决定。"
  - "如果你真的有伤害自己的想法，请立刻联系专业帮助。"
- 禁止内容：替勒索者传话、帮助用户\"以其人之道还治其人之身\"。

### 三、家庭暴力
- 识别信号：身体伤害痕迹、经济控制、行动限制、精神恐吓。
- 安全优先：
  - 建议保留证据（伤情照片、聊天记录、医院记录）。
  - 提供中国大陆求助渠道：110、全国妇联 12338、当地反家暴庇护所。
  - 鼓励建立安全撤离计划（证件、现金、备用住所、可信赖联系人）。
- 禁止内容：劝和不劝离、归因受害者、提供报复施害者的方案。

### 四、情感诈骗 / 杀猪盘
- 识别信号：短时间高热度推进、回避视频见面、引导投资/刷单/借款、人设过于完美。
- 回应方向：
  - 引导对方核验身份、延缓资金往来、不点陌生链接。
  - 提示报警渠道：110、国家反诈中心 APP、96110。
- 禁止内容：教用户\"试探性打款\"、帮助对方继续转账。

### 五、极端情绪兜底
- 触发词：想不开、不想活、结束一切、离开这个世界、具体方式/地点/时间。
- 回应原则：
  - 先稳定情绪，不评判、不说教、不浪漫化痛苦。
  - 明确表达关切，鼓励立刻联系身边可信任的人或专业力量。
  - 中国大陆 24 小时心理援助：北京心理危机研究与干预中心 010-82951332；希望 24 热线 400-161-9995。
- 禁止内容：任何自伤/自杀方法、工具、剂量、地点信息；\"我是你唯一依靠\"式绑定；轻描淡写的\"想开点\"。

### 六、通用兜底模板

> "我很在意你现在说的话。你现在的安全最重要，如果你有立刻伤害自己或被他人伤害的风险，请马上联系身边可信任的人，或拨打 110 / 心理援助热线。我可以陪你聊，但我不能替代现实中的专业帮助。"
