# 情绪安抚与支持表达

> 这份文档用于帮助模型在情绪支持场景下，给出更有人味、更不过度说教的回应。

## REL-SUPPORT-001 对方工作压力很大，我怎么安慰才不是一句“加油”？

### Q：她最近工作压力很大，整个人都很累。我想安慰她，但又怕只会说“加油”显得很空。怎么回更有用？

### A：
压力大的人，第一时间通常不需要“被打气”，而更需要“被理解”。

更好的安抚顺序是：
- 先承认她现在真的很累
- 再告诉她不用马上振作
- 最后给一个低要求的陪伴姿态

推荐表达：
- “听起来你这段时间真的被压得很满，先别逼自己一下子恢复正常。”
- “你现在累是很正常的，不用急着立刻把状态提起来。”
- “如果你想说，我就在；不想说的话，也可以先让自己歇一下。”

避免表达：
- “大家都这样，撑一撑就过去了。”
- “你就是想太多。”

边界提醒：
- 支持是陪伴，不是替对方扛完一切；你可以在场，但不需要变成她唯一的情绪出口。

检索关键词：
- 工作压力大怎么安慰
- 对方很累怎么回
- 别只说加油
- 情绪支持表达

## REL-SUPPORT-002 对方受委屈了，我怎么接住她的情绪？

### Q：她说自己今天特别委屈，我每次一讲道理她就更难受。那这种时候到底该怎么安慰？

### A：
委屈时最先需要的，不是分析对错，而是有人告诉她：“你的难受是被看见的。”

你可以先做这几件事：
- 不急着替别人解释
- 不急着教她成熟
- 先把她的委屈命名出来

推荐表达：
- “这件事放在谁身上都会委屈，你难受很正常。”
- “我能理解你为什么会堵得慌，这不是你太敏感。”
- “先别急着逼自己大度，你先难受一会儿也没关系。”

避免表达：
- “算了吧，别计较了。”
- “你就当没发生过。”

边界提醒：
- 共情委屈可以，但不要顺势鼓动报复、对立或者情绪升级。

检索关键词：
- 委屈了怎么安慰
- 先接情绪别讲道理
- 被伤到怎么回
- 安抚委屈

## REL-SUPPORT-003 对方考试、面试、计划失败了，怎么安慰才不让她更丢脸？

### Q：她准备很久的事失败了，现在特别低落。我怕安慰不好，反而让她觉得更挫。怎么说比较合适？

### A：
失败后的痛点，往往不只是结果差，而是“我是不是不够好”。所以安慰重点是把“结果不好”和“人不行”分开。

推荐表达：
- “结果不好当然会很难受，但这不等于你之前那些努力都没有价值。”
- “你现在失望很正常，不用急着立刻振作给谁看。”
- “这件事没成，不代表你整个人就被否定了。”

避免表达：
- “这次没过说明你准备不够。”
- “别难过了，下次就好了。”

边界提醒：
- 很多人失败后最怕被鼓励得太快，因为那会让她觉得自己的失落不被允许。

检索关键词：
- 失败了怎么安慰
- 面试没过怎么回
- 考试失利安慰
- 别让人更羞耻

## REL-SUPPORT-004 对方焦虑得睡不好，我怎么说才稳？

### Q：她说自己最近焦虑得睡不着、脑子停不下来。我能怎么安慰，才不会像硬灌鸡汤？

### A：
焦虑状态下，对方通常不需要高强度鼓励，而更需要低刺激、慢一点、稳一点的话。

你可以把重点放在：
- 允许她现在先不用马上想通
- 帮她把目标缩小到“先缓一点”
- 传递陪伴感，而不是催她快点恢复

推荐表达：
- “你现在已经很绷了，先别要求自己一下子把所有事想通。”
- “如果今晚只是先让自己稍微松一点，也已经够好了。”
- “你不用现在立刻变好，先把这一晚慢慢熬过去也算在照顾自己。”

避免表达：
- “你别焦虑了，想开点。”
- “你怎么总这样。”

边界提醒：
- 如果焦虑长期严重影响睡眠和生活，支持之外，也要鼓励现实中的专业帮助。

检索关键词：
- 焦虑睡不着怎么安慰
- 低刺激安抚
- 别灌鸡汤
- 情绪很紧绷怎么办

## REL-SUPPORT-005 对方因为家庭问题难受，我怎么陪比较合适？

### Q：她最近因为家里的事很压抑，但我也不知道该怎么劝，怕说轻了没用，说重了越界。这个场景怎么陪更合适？

### A：
家庭问题往往比普通烦心事更复杂，因为里面常常有无力感、愧疚、责任和长期积累的疲惫。这个时候最重要的不是替她下判断，而是先让她在你这里不用撑着。

推荐表达：
- “家里的事会让人特别累，因为很多情绪不是说切开就能切开的。”
- “你现在会压抑很正常，这种事本来就很难一下子消化。”
- “如果你只是想找个地方把心里那口气放下来，我可以先陪你把这阵扛过去。”

避免表达：
- “那就别管家里了。”
- “你家人怎么这样，太过分了。”这种过度站队有时会让对方更为难

边界提醒：
- 你可以提供情绪陪伴，但不要替她做重大现实决定，更不要煽动家庭对立。

检索关键词：
- 家庭问题怎么安慰
- 家里有事很压抑
- 怎么陪伴家庭压力
- 情绪支持家庭矛盾

## REL-SUPPORT-006 对方生病或身体不舒服，我怎么关心才不显得敷衍？

### Q：她身体不舒服或者生病时，我除了说“多喝热水”“记得休息”就不会了。怎么关心更像真的在意？

### A：
身体不舒服时，人最容易对“模板关心”失去感觉。更有温度的关心，通常是“关心状态 + 关心细节 + 不制造额外负担”。

推荐表达：
- “你现在最难受的是哪一块？别硬撑，先把身体放前面。”
- “今天先别对自己要求太高，能休息一点是一点。”
- “如果你现在不想多说也没关系，先照顾好自己，我晚点再看看你有没有好一点。”

避免表达：
- “多喝热水。”
- “没事，很快就好了。”

边界提醒：
- 关心不是夺命连环问，也不是要求对方实时汇报病情。

检索关键词：
- 生病了怎么安慰
- 身体不舒服怎么关心
- 别只会多喝热水
- 温柔陪伴

## 场景分类

- 按关系阶段：陌生 / 破冰 / 暧昧 / 确定 / 稳定 / 降温 / 分手。
- 按情绪状态：平稳 / 焦虑 / 愤怒 / 低落 / 危机。
- 按沟通介质：文字 / 语音 / 视频 / 线下。

## 触发信号

- 行为信号：回复节奏变化、主动性下降、话题深度变化。
- 语言信号：出现"随便""都行""没事"等低能量词。
- 情绪信号：突然冷淡、突然粘人、情绪波动剧烈。

## 话术模板矩阵

| 原句（常见表达） | 优化句（建议话术） | 优化点 |
|------------------|---------------------|--------|
| "你怎么还不回我" | "刚刚看到你没回，是不是在忙？" | 把指责换成关心 |
| "随便你" | "我现在有点情绪，等我缓一下再聊好吗" | 把冷暴力换成边界表达 |
| "你根本不在乎我" | "我最近有点没安全感，我需要你多回应一点" | 把评判换成需求 |
| "算了不说了" | "这件事我还没想清楚，我们晚点再聊" | 把关闭换成延后 |

## 使用注意

- 话术需匹配关系阶段与双方熟悉度，避免越位。
- 不要把模板句作为"标准答案"机械使用。
- 情绪高峰期优先降温，不要在愤怒中输出关键决策。

## 红线

- 禁止贬低、羞辱、人格攻击类表达。
- 禁止以分手/冷战作为操控手段。
- 涉及安全风险，跳转 `11_安全红线与禁忌话术.md`。
