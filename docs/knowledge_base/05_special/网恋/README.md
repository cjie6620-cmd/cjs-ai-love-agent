# 网恋专题（Online Romance）

> 网恋是**通过网络渠道（社交软件、相亲 App、游戏社区、内容平台等）建立并发展的亲密关系**。Statista 数据显示 2023 年全球超过 3.5 亿人使用过婚恋类 App。网恋既放大了"匹配效率"，也放大了"信息不对称"。

---

## 一、渠道类型与特征

| 渠道 | 典型平台 | 优势 | 风险 |
|------|----------|------|------|
| 婚恋 App | 探探、Soul、Bumble、Hinge | 目标明确、匹配高效 | 形象美化、资料失真 |
| 兴趣社群 | 豆瓣、B 站、小红书、Discord | 价值观筛选 | 关系发展慢、地域远 |
| 游戏社区 | MMO、Moba、开黑平台 | 共同体验多 | 角色人格 ≠ 现实人格 |
| 内容平台 | 评论区、直播、播客 | 认知前置 | 粉丝滤镜、权力不对等 |

## 二、关键知识点

### 2.1 线上聊天节奏
- **前 3 次互动**：轻松共鸣 + 轻自我披露（兴趣、生活状态）。
- **第 4–10 次**：增加深度披露（价值观、成长经历），注意互惠性。
- **切换媒介**：文字 → 语音 → 视频，符合**媒介丰富度递进原则**，避免过早/过晚切换。

### 2.2 真实性识别
- 要求对方**至少 1 次视频验证**，不接受以"社恐/镜头恐惧"为由长期拒绝。
- 用**时间戳验证法**：临时要求拍一张"手比数字 3"的照片。
- 警惕**完美人设**：照片极度精修、朋友圈仅三天可见、拒绝任何现实细节。

### 2.3 奔现策略
- **中立地点**：首次见面选城市中心公共场所，白天为佳。
- **时间可控**：2–3 小时咖啡为宜，不要一见面就安排过夜行程。
- **告知好友**：把对方照片、联系方式、见面地点、预计回家时间告知至少一位好友。
- **心理准备**：线下真人与线上人设存在落差是常态，留出 2–3 次见面判断期。

### 2.4 长期维系
- 把网恋当作"异地恋 + 信息有限"的叠加挑战，参见 `05_special/异地恋/README.md`。
- 尽早制定**见面周期 + 长期同城计划**。

## 三、诈骗与安全红线

⚠️ 以下行为**几乎 100% 是诈骗/灰产**，请立即终止并报警：

- 对方主动提出**投资、刷单、带你赚钱**。
- 以**手术费、机票、卡关、海关**为由索要金钱。
- 长期拒绝视频但反复索要礼物、转账、会员卡。
- 让你下载非官方商店的 App、点击陌生链接完成"充值"。
- 自称**军人、工程师、海外高管**，却从不能核实身份。

参见并向用户推送：`relationship_knowledge/11_安全红线与禁忌话术.md`。
