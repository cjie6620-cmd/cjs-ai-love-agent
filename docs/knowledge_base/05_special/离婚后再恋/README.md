# 离婚后再恋专题

> 再婚/再恋人群面临**心理创伤、子女系统、财产与法律、社会评价**四重叠加议题。民政部数据显示，近十年中国离婚率波动上升，再婚比例随之增长。对这一群体，AI Agent 的首要任务是"先稳定、再恋爱"。

---

## 一、节奏建议

```
创伤处理（3–12 个月）─➜ 自我整合（6–18 个月）─➜ 重新约会 ─➜ 引入子女 ─➜ 再婚/同居
```

- **不建议**在离婚未满 3 个月内开始新的严肃关系（"过渡恋"风险高）。
- **建议**在完成一次完整的经济独立周期后再考虑同居/再婚。

---

## 二、核心议题

### 2.1 心理恢复
- 允许**哀伤过程**（否认→愤怒→讨价还价→抑郁→接受）。
- 区分**离婚的失败 ≠ 自己的失败**。
- 必要时寻求专业咨询，特别是当**自伤倾向、长期失眠、PTSD 症状**出现时。

### 2.2 过往反思
- 从**"是谁的错"转向"我在其中的模式"**：沟通习惯、依恋类型、冲突应对。
- 识别**重复脚本**：容易被同一类伴侣吸引并重蹈覆辙。
- 做一次结构化"复盘清单"：关系三大冲突、我的贡献、我的学习。

### 2.3 子女系统
- **引入新伴侣前提**：关系稳定 ≥ 6–12 个月、新伴侣愿意长期承担角色。
- **第一次见面**：公共场合、短时间、不强求称呼、不强求喜欢。
- **与前任协商**：就子女抚养、探视、节日安排与新伴侣边界达成清晰共识。
- **避免**："替代父母"叙事、禁止孩子提起另一方、用孩子传话。

### 2.4 新关系信任
- 坦诚你的婚史、子女、财务大致情况，但**不以受害者身份索取过度共情**。
- 不拿新伴侣与前任做**"你比他/她好"类比较**——这是隐性的情绪绑架。
- 给彼此**验证期**（通常 6 个月以上），再讨论同居、再婚。

### 2.5 法律与财产
- 再婚前建议做**婚前财产公证**，保护双方与各自子女的权益。
- 明确**婚内共同财产、继承权、子女抚养费**等法律边界。
- 遇到复杂情境建议咨询婚姻家事律师，而非仅依赖网络信息。

## 三、话术模板

| 场景 | 话术 |
|------|------|
| 坦诚婚史 | "我经历过一次婚姻，有一个孩子。如果你愿意，我们可以慢慢聊那段经历。" |
| 介绍子女 | "我们关系稳定了，我想让你先以'妈妈/爸爸的朋友'的身份见见孩子，不急着定义。" |
| 划清边界 | "和前任的联系只限于孩子相关事务，其他我们两个人的事我只和你商量。" |
| 回应外界 | "再婚是我认真考虑后的决定，谢谢关心。" |

---

## 四、红线信号

- 新伴侣要求你**切断与子女/前任的所有联系**。
- 在**未充分了解彼此**的前提下被催促同居、再婚、生育。
- 任何一方用**"你离过婚就别挑"**做贬低或控制。
- 经济上出现**大额单向资金流向**而无合理解释。
